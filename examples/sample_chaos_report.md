## Notes

This report was generated by running Dock Chaos against the
[Distributed URL Shortener](https://github.com/AaditTrivedi/url-shortener)
project with `--intensity high --duration 120`.

**Key finding during development:** Before implementing the Redis circuit breaker,
`container_kill` on the Redis container caused a 12-second latency spike in the
URL shortener app. The app was waiting for the default 30-second Redis connection
timeout before falling back to PostgreSQL-only mode.

**Fix applied:** Reduced Redis socket timeout to 1 second, added a circuit breaker
that opens after 3 consecutive failures and skips Redis entirely for 30 seconds.
After the fix, recovery from Redis failure dropped from ~12,000ms to under 200ms.
